# 🎧 DJ & Mixing Zone - Wirtualny Launchpad

Prosty, interaktywny projekt strony internetowej dla pasjonatów muzyki, miksowania i DJ-ingu. Strona zawiera podstawowe informacje oraz wbudowany, działający minikontroler (Launchpad), który pozwala na generowanie dźwięków bezpośrednio w przeglądarce.

## 🚀 Live Demo
Możesz przetestować działającą stronę tutaj:
👉 [ZOBACZ PROJEKT ONLINE](https://krystian57350.github.io/MojaStrona/)

## 🛠️ Użyte technologie
* **HTML5** – struktura strony i elementy Launchpada
* **CSS3** – nowoczesny, mroczny design z neonowymi efektami w stylu klubowym
* **JavaScript** – obsługa interakcji i generowanie dźwięków (Web Audio API)

## 🎛️ Funkcje projektu
* **Syntetyczne dźwięki:** Gra bez zewnętrznych plików audio – dźwięki (Kick, Snare, Hi-Hat, Synth) są generowane na bieżąco przez kod JS.
* **Responsywność:** Strona dopasowuje się do ekranów smartfonów i komputerów.
* **Hosting:** Projekt jest hostowany całkowicie za darmo za pomocą usługi GitHub Pages.
